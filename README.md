# Arduino ESP8266 Control Relay with Telegram-Bot

### Required Components:
- Arduino UNO
- ESP01 module (ESP8266)
- Relay 5v
- Male & Female connectors
- Breadboard mini 8.5x5.5 cm 400 holes

### Required Libraries
- SoftwareSerial
- ESP8266WiFi
- WiFiClientSecure
- UniversalTelegramBot
- ArduinoJson
